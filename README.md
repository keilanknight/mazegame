# Maze Game

A simple maze game demo for the ZX Spectrum, written to demonstrate how easy it is to create games with [Boriel BASIC](https://zxbasic.readthedocs.io/).

Each level generates a new random maze. Collect all the diamonds, then reach the exit to continue to the next stage.

## Controls

- `Q` — up
- `A` — down
- `O` — left
- `P` — right
- `Space` — restart

Compile `mazegame.bas` with the Boriel BASIC compiler, then load the resulting program in a ZX Spectrum emulator or on real hardware.
